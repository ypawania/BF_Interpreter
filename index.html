Esoteric Language Interpreter

brain is an esoteric programmming language. It has only 8 keywords:
  • < and > Move the pointer to the left and right
  • + and - increment and decrement the pointer by 1
  • . and , are for IO. A period will output the character with the ASCII value of the active cell and the comma will input the opposite
  • [ and ] are control flow operators. The code between them is executed as long the value of the active cell is not 0 when the closing bracket is evaluated
